# crypto-diff

Compare real-time crypto prices and trades with percentage-based insights

## Tech Stack

- [Preact (with signals)](https://preactjs.com/)
- [MantineUI](https://mantine.dev/)
- [Tanstack Query](https://tanstack.com/query/)
- [Nivo Charts](https://nivo.rocks/)
- [Alpaca API (REST and WS)](https://alpaca.markets/)
